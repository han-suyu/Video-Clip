# Video-Clip
基于FFmpeg的视频剪辑小工具

<font color='blue'>步骤：导入视频 -> 选择开始时间和终止时间 -> 修改输出文件名(可省略) -> 执行剪辑 -> 目标文件所在的文件夹自动打开</font>
